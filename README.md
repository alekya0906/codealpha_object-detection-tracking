# Real-Time Object Detection and Tracking using YOLOv8

## Overview
This project performs real-time object detection, tracking, and counting using YOLOv8 and OpenCV.

## Features
- Real-time object detection
- Object tracking with IDs
- Object counting
- Video output saving
- Webcam support

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics

## How to Run

```bash
pip install ultralytics
pip install opencv-python
pip install numpy

python main.py
```

## Output
The project generates an output video file containing detected and tracked objects.
